### :newspaper: 프로젝트 소개
각 지역별로 누구나 알 수 있는 선수들을 자랑하고 서로 구경하고 응원함으로써 축구를 잘 알지 못하더라도 쉽게 유명선수들을 찾을 수 있도록 공유하는 플랫폼입니다.

### :raising_hand: 팀원
- 최형용
- 김진환
- 김민섭
- 최한솔

### :floppy_disk: Framework
- Front-end: HTML, CSS, JavaScript
- Back-end: Flask
- Database: mongoDB

### :rocket: Modules

- PyJWT
- dnspython
- requests
